术语
1、XML；
         可扩展的标记语言，用于定义  文档   结构的格式语言
        （用标记的方式分割文档，使计算机识别）
      eg:
        语言：每周一书写文章
        规定计算机识别格式：
        <任务>
          <执行时间>每周一</执行时间>
          <执行方式>书写</执行方式>
        <任务>
2、HTML：
        超文本标记语言
        超文本：有跳转链接
        标签：<></>
        由w3c组织定义的语言标准，是用于描述   页面   结构的语言
        （页面有什么东西；某个标签表示什么含义）
